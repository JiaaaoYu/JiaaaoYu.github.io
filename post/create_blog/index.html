利用Gridea和GitHub Pages快速免费搭建个人博客

2020-06-25 4 min read # tools

本文介绍如何利用Gridea和Github Pages快速搭建属于自己的个人博客，配置简单，操作方便，维护轻松，界面简洁却又不失优雅，更没有像CSDN，简书和博客园那样的牛皮癣广告。

为何搭建个人博客

  1. 采坑记录：你研究好久才解决的问题，下次遇到相同问题时候，可以快速找到解决方法；别人在遇到同样问题时，也可以参照你的博客快速解决；
  2. 成就感：将自己所学所想，所思所感分享与他人，当自己的博客帮助让他人有所成长时，并获得他们的评论或者点赞，成就感简直爆棚；
  3. 梳理知识体系：写博客是一个长期的积累过程，让你定期复盘并梳理知识结构，不仅对学习有很大帮助，还可以作为一个非常重要的面试加分项；
  4. 锻炼总结和表达能力：能把自己所想准确简练地表达出来，这些软实力在跨部门沟通与协作，向上汇报工作以及在晋升答辩的时候都是非常重要的；
  5. 打造个人影响力：通过写个人博客，帮助他们解决问题并获得认同，在有一定影响力之后，无论在物质还是精神层面都有不错的收获。

为何选择Gridea

目前大家常用的几个写博客站点：博客园，CSDN和简书等，这些网站简单易用，有很大的程序员用户群体，但像我这种有轻微强迫症的人，在看到：


它们之后，这多如牛毛的广告和杂乱的布局让我实在没有写博客并继续维护的欲望；对比来看，我利用Gridea搭建的个人博客，简洁却又不失优雅，配置简单，后续维护更是简单，也让我有了继续更新和维护的欲望：

我用Gridea搭建的个人博客： Chiaao

如何搭建个人博客

Github Pages配置

1. 新建repository

登录github，在右上角加号这里，新建repository(New repository)

2. 设置repository

repository设置成username.github.io形式，例如我这里是username：JiaaaoYu，设置成：JiaaaoYu.github.io，Github会开启同名的GitHub Pages：

下一步后确认是否开启，未开启手动操作即可：

登录上面的网址，看是否能打开网站

3. 创建Token

点击右上角+号，点击setting，然后点击左侧Developer settings

然后点击Personal access tokens，点击Generate new token

4. 配置Token


把access tokens这一串字符复制下来，待会儿在配置Gridea的时候会用到

Gridea配置

1. 下载Gridea

Gridea 是一个静态博客写作客户端，帮助你更容易地构建并管理博客下载Gridea

2. 配置Gridea

Gridea配置按下图指引配置即可

3. 配置评论

如果想开启评论的话，回到上面的github设置Developer settings界面

注册并配置OAuth

回到Gridea，填写Client ID，Client Secret

至此，所有的配置全部结束，然后点击Gridea左下角同步，待同步成功之后，重新打开网站，界面就变成了Gridea为我们初始化的Blog

写Blog

所有配置完成之后，我们就可以开始自己的Blog之旅了，所有的文章都可以在这里新建和编写，推荐使用markdown格式编写，本文就是通过markdown编写，利用Gridea发布在个人网站上，Gridea的常用操作以及如何使用markdown，后文继续介绍。
